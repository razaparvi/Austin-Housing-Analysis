# Austin-Housing-Analysis
Austin Housing Price Predictor
An End-to-End Machine Learning Project

 Executive Summary
This project analyzes 15,000+ housing listings in Austin, TX, to predict property values using a Random Forest Regressor. By incorporating geospatial data (Latitude/Longitude) and school ratings, the model achieved an R² score of 0.71.

 The Tech Stack
Python: Pandas, NumPy, Scikit-Learn.

Visualization: Matplotlib, Seaborn, Folium (Interactive Heatmaps).

Modeling: Random Forest Regression with Log-Transformation.

💡 Key Insights
Location is King: Adding GPS coordinates improved model accuracy by 10% compared to basic features alone.

Price Drivers: Square footage and school quality were the top two predictors of value.
